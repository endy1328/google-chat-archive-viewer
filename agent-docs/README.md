# agent-docs 개요

`agent-docs`는 AI 에이전트와 하네스가 이 저장소에서 일관되게 작업하기 위한 운영 문서를 모아두는 디렉토리다.

## 포함 문서

- `harness.md`: 하네스 기반 작업 흐름과 체크리스트
- `documentation.md`: 문서 작성, 갱신, 리뷰 원칙
- `git-workflow.md`: 초기 git 브랜치 전략과 운영 규칙 초안

## 사용 원칙

- 저장소 규칙은 `../AGENTS.md`를 기준으로 한다.
- 반복적으로 발생하는 작업 절차는 이 디렉토리에 추가한다.
- 특정 기능 전용 문서가 필요하면 기능명 기준으로 파일을 추가한다.

## 초기 운영 방식

- 프로젝트 구조나 실행 방법이 확정되지 않은 문서는 "초안" 상태로 표시한다.
- 초안 문서는 실제 코드, 스크립트, 배포 흐름이 정해질 때 바로 갱신한다.
- 문서가 코드보다 먼저 만들어졌다면, 검증되지 않은 내용은 예시로만 적고 사실처럼 단정하지 않는다.
